AutoComplete with caching behavior

This component is the Wicket Autocomplete component with cache behavior added. The cache has a expiration time. Both the cache and its expiration time can be configured in AutoCompleteSettings.java through its setCacheDuration(int) method.

Example

ChoiceSearchBoxBehavior

The ChoiceSearchBoxBehavior adds a search field which applies search criteria to a given AbstractChoice. Note: This behavior cannot be reused between components.
